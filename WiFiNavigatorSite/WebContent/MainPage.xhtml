Приложение для смарфонов

Вы можете загрузить демо-версию приложения для своего android смартфона прямо сейчас!. Это позволит вам загружать и пользоваться картами, расположенными на нашем сайте. Для скачивания нажмите на кнопку загрузки, расположенную внизу этой панели.

Загрузка карт

Вы можете загрузить карты для навигатора, нажав на кнопку, расположенную ниже. Для этого воспользуйтесь встроенным в навигатор сканером QR-кодов. Просто наведите камеру на выбранный код и карта загрузится на телефон. (пока количество карт ограничено)

Инновационное навигационное приложение

Приложение Wi-Fi Navigator обладает следующими преимуществами:

  • Работает внутри зданий
  • Не имеет аналогов в России
  • Преимущество №3
  • Преимущество №4
  • Преимущество №5
  • Преимущество №6

Наше навигационное приложение работает внутри зданий помеченных специальным значком Wi-Fi Navigator. Вы можете использовать приложение для того, чтобы найти интересующий вас объект и проложить к нему маршрут.
